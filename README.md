# green-credit-agricultural-carbon-reduction
This repository provides the data and code used in the empirical analysis of this study to enhance research transparency and reproducibility. The uploaded materials include: (1) spatial weight matrix data, including the green technology composite geographic matrix, the geographic distance matrix, and the economic composite geographic matrix; (2) the processed dataset used in the empirical analysis; and (3) the relevant code for constructing the geographic matrix, constructing the composite matrix, and conducting Moran's I test. These materials can be used to replicate the construction of spatial weight matrices, the spatial autocorrelation test, and the subsequent empirical analysis in this study.
